Inventory Restocking Decision System - OpenEnv
A complete OpenEnv implementation for AI agent-based inventory optimization. This dashboard simulates a multi-SKU warehouse where agents (or humans) learn to make optimal restocking decisions while minimizing holding costs and preventing stockouts.

🚀 Quick Start
Prerequisites
Python 3.10+
HuggingFace account (for model API access)
GitHub account
1. Local Setup
# Clone the repository
git clone <your-repo-url>
cd inventory-management-env

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.tx
